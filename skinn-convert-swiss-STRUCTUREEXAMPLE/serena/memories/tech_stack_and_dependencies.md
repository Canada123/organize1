# Tech Stack and Dependencies

## Core Technology Stack

### Frontend Framework
- **React 18** - Latest stable version with concurrent features
- **TypeScript 5** - Strict mode enabled, explicit typing required
- **Vite** - Fast build tool and development server
- **React Router DOM 6** - Client-side routing with language prefixes

### Styling and UI
- **Tailwind CSS 3** - Utility-first CSS framework
- **shadcn/ui** - Component library based on Radix UI primitives
- **Radix UI** - Accessible component primitives (extensive collection)
- **Lucide React** - Icon library
- **Framer Motion** - Animation library for enhanced UX

### State Management and Data
- **Context API** - Global client state management
- **TanStack React Query 5** - Server state management and caching
- **Supabase Client** - Database and authentication integration

### Forms and Validation
- **React Hook Form 7** - Form state management
- **Zod 3** - Runtime type validation and schema definition
- **@hookform/resolvers** - Form validation integration

### Development and Testing
- **Vitest** - Unit testing framework
- **Playwright** - End-to-end testing
- **ESLint 9** - Code linting
- **TypeScript ESLint** - TypeScript-specific linting rules

### Specialized Libraries
- **date-fns** - Date manipulation utilities
- **class-variance-authority** - Utility for component variants
- **clsx & tailwind-merge** - Conditional class name utilities
- **react-helmet-async** - Document head management
- **next-themes** - Theme management system

## Development Environment
- **Node.js 18+** required
- **npm 9+** for package management
- **Darwin (macOS)** primary development platform
- **Port 8080/8081** for development server
