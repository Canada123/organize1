# Content Documentation

## Overview
Management of website content including translations, marketing copy, and user-facing text across 4 languages.

## Content Structure

### Translation Files
Located in `docs/content/translations/`:
- `en.md` - English content
- `de.md` - German content
- `fr.md` - French content
- `it.md` - Italian content

### Content Categories
- **Marketing Copy**: Homepage, landing pages
- **UI Labels**: Buttons, forms, navigation
- **Error Messages**: Validation, system errors
- **Legal Content**: Terms, privacy, consent
- **Email Templates**: OTP, confirmations

## Translation Management

### Process
1. English content created first (master)
2. Professional translation to other languages
3. Review by native speakers
4. Technical review for consistency
5. Implementation in code

### Quality Checks
- Terminology consistency
- Cultural appropriateness
- Legal compliance
- Character encoding
- Text length variations

## Content Guidelines

### Tone of Voice
- Professional yet approachable
- Clear and concise
- Empathetic for health topics
- Action-oriented CTAs

### Swiss Specifics
- Use formal address (Sie/Vous/Lei)
- Reference Swiss healthcare system
- Include canton-specific information
- Use CHF for currency

## Update Process

### Content Changes
1. Update English master file
2. Mark for translation
3. Update other language files
4. Test in application
5. Deploy with version tag

### Version Control
- Track changes in Git
- Tag releases with content versions
- Maintain changelog
- Archive old versions

## SEO Optimization

### Meta Content
- Page titles (60 chars max)
- Meta descriptions (160 chars max)
- Open Graph tags
- Structured data

### Keywords
- Swiss German variations
- Healthcare terminology
- Local search terms
- Insurance provider names

## Legal Requirements

### Compliance
- GDPR consent text
- Swiss data protection
- Medical disclaimer
- Terms of service
- Cookie policy

### Updates
- Regular legal review
- Compliance tracking
- Update notifications
- User consent refresh
